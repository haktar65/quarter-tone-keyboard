# Steinway-Like Archive Index

This directory contains the full documentation snapshot of the archived piano-like intermediate branch.

Terminology note:

- `VT` in this archive is the older quarter-tone-key label and corresponds to the active `QT` family naming
- this archive predates the active four-family split `MWT`, `MST`, `QWT`, `QST`
- where the archive distinguishes `VT` rows above whole tones vs. semitone keys, those map conceptually to active `QWT` and `QST`

Archived files in this directory:

- `Research.md`
- `BuildGuide.md`
- `MP5GeometryReference.md`
- `SteinwayLikeArchive.md`

Interpretation:

- `SteinwayLikeArchive.md` is the compact branch summary
- `Research.md` and `BuildGuide.md` are the full archived working documents from that branch
- `MP5GeometryReference.md` preserves the MP5-related geometry note that belonged to the same reference direction

This archive snapshot is intentionally kept separate from the active synth / semi-weighted workstream.
